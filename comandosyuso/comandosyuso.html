Informacion sobre los comando de git y el uso de GitBash

pwd: Sirve para poder ubicar en qué directorio me encuentro
cd: Sirve para situarse en un directorio
cd ..: Subir un nivel de directorio
is: Muestra en forma de lista todo lo que se encuentra dentro del directorio donde nos encontramos.
clear: Sirve para borrar la pantalla
mkdir: Sirve para poder crear directorios
touch: Sirve para poder crear ficheros de archivos
exit: Sale de la sesion git

git init: • Todo lo que se escriba dentro de la carpeta se encontrará dentro de git.
• Acá se guarda el historial de versiones y se encuentra oculto dentro de la carpeta\
• Instrucción que se ejecuta solo una vez
git add: • Lleva el control de archivos que se agregan.
cd ..: • Subir un nivel de directorio
git commit: • Comando que indica que esta lista alguna funcionalidad para ser una versión de código.
git commit –m “Comentario”: • Archivos enviados al repositorio añadiendo los comentarios de los cambios
git remote add origin "URL": • Especifica el origen de nuestro repositorio. La URL se obtiene del repositorio de GitHub.
• Ejemplo: git remote add origin https://github.com/pedroarmas24/hola-mundo.git
git status: • Permite ver lo que tenemos en el archivo, este sirve para ver si hay cambios o no por subir al repositorio (si es necesario crear un commit o no)
git commit –am “Comentario descriptivo”: • Sirve para enviar archivos al repositorio, es decir, los cambios realizados
git push: • Envía la rama especificada a una "remote", junto con todos los commits y objetos internos necesarios. De este modo se crea una rama local en el repositorio de destino
git push origin master: • Es el nombre que se le coloca al repositorio remoto al que se conecta
git pull: • Se emplea para extraer y descargar contenido desde un repositorio remoto y actualizar al instante el repositorio local para reflejar el contenido.
Branch: • Es una rama diferente a la rama master y se utiliza de forma colaborativa. Con ella podremos asignar tareas y ordenar de mejor manera nuestro proyecto.
Merge: • Es una fusión de ramas
